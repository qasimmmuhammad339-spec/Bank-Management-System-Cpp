# Bank Management System (C++)

A simple console-based Bank Management System written in C++.
This project is created to practice Object-Oriented Programming (OOP)
concepts and basic programming logic.

## Features
- Create a bank account
- Deposit money
- Withdraw money
- Check account balance
- Display account details
- Menu-driven program

## Concepts Used
- Classes and Objects
- Constructor
- Encapsulation
- Input validation
- Loops and switch-case

## How to Run
- Compile the program using any C++ compiler
- Run the program
- Follow the on-screen menu
